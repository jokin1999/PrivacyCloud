# Privacy Cloud

*针对极客用户开发的私有云对象存储管理方案*

---

## 版本信息解释

*提供的版本信息存在版本差异，缺少为正常现象*

- `auto_update`： 是否支持自动更新
- `auto_update_support`： 自动更新支持版本
  - `0`：只要有自动更新功能即可更新
  - `其他`： 从这个版本起都可以进行自动更新
- `auto_update_except`： 自动更新除外列表
- `level`： 详情请查询 [参考页](./i_level.md)
- `necessity`： 必要性，0~9的区间值，数值越大越必要。
- `releastAt`： 发布日期
- `is_build`： 是否为测试版

*微功能：无需手动运行的功能。*

*非主程序：如SDK等第三方支持的程序。*

*`level`可能由多个数字组成，以表达更准确。*
